Manage launchd services.

## Functions

### `launchd.service`

Manage the state of launchd managed services.

```python
launchd.service(
    service,
    running=True,
    restarted=False,
    command=None,
)
```

<ParamField name="service" type="str" required>
  Name of the service to manage
</ParamField>

<ParamField name="running" type="bool" default="True">
  Whether the service should be running
</ParamField>

<ParamField name="restarted" type="bool" default="False">
  Whether the service should be restarted
</ParamField>

<ParamField name="command" type="str" default="None">
  Custom command to pass like: `launchctl <command> <service>`
</ParamField>

## Examples

```python
from pyinfra.operations import launchd

# Manage a launchd service on macOS
launchd.service(
    name="Ensure ssh is running",
    service="com.openssh.sshd",
    running=True,
)
```
